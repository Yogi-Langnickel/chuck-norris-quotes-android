Welcome to Quote Batttle Royal! This application fetches random Chuck Norris quotes, cat facts, and dog facts from public APIs, then displays them in a user-friendly interface.

With a clean and modern design, the app utilizes Jetpack Compose for a responsive UI and follows the MVVM architecture for maintainability and scalability.

Key Features:
- Fetch random Chuck Norris quotes with a simple tap.
- Fetch random cat facts from rotating public sources.
- Fetch random dog facts from Dog API.
- Start a quote battle, swipe away the loser, and track daily, weekly, and monthly local scores.
- Bring in the next challenger from a random non-winner fact stream, with fallback if an API fails.
- Enjoy smoother battle swipes with spring settling, continuous loser exit motion, and no card tilt.
- See Chuck, Cat, and Dog celebrate with source-specific victory animations when they win.
- Follow the current champion streak as one source keeps defending the battle arena.
- Switch between dedicated Battle Mode, Chuck Facts, Cat Facts, and Dog Facts tabs.
- Open the latest GitHub release from a visible Update app button for easier manual updates.
- Copy or share the current quote or fact.
- Beautifully designed UI using Jetpack Compose.
- Built with Kotlin for a seamless Android experience.

Join us in celebrating the legendary Chuck Norris with this fun and entertaining app!
